# Ejercicio 6

## Compilación

```sh-session
$ make
```

## Ejecución

```sh-session
-- Crea un servidor con 3 threads:
$ ./time_server_multi 0.0.0.0 2222

-- Crea un servidor con 2 threads:
$ ./time_server_multi 0.0.0.0 2222 2
```

> **Nota**: el [ejercicio 3](../ejercicio3) se puede utilizar para interactuar
con el servidor.
